# LinguaCore Redesign — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Rediseñar LinguaCore de un dark UI genérico a "The Fluid Scholar" — light mode con paleta violeta/lila, tipografía Plus Jakarta Sans + Be Vietnam Pro, componentes rediseñados y sistema de temas Pro.

**Architecture:** Design system first (tokens CSS en Tailwind v4 `@theme`), luego páginas de mayor impacto (Landing → Dashboard), luego resto de páginas. Los temas Pro se implementan como `data-theme` en `<html>` con CSS custom properties. No se toca la lógica de negocio ni los agentes AI — solo UI.

**Tech Stack:** React 19, Vite 7, Tailwind CSS v4 (via `@tailwindcss/vite`), Framer Motion, React Router 7, Supabase (para persistir tema Pro)

**Design Reference:** `docs/plans/2026-03-25-redesign-design.md` — leer antes de implementar cualquier tarea.

**Step 2: Reemplazar el ThemeContext**

El nuevo ThemeContext debe manejar: `mode` (light | dark) + `proTheme` (purple | ocean | forest | midnight).

```tsx
import { createContext, useContext, useEffect, useState, ReactNode } from 'react'

type ColorMode = 'light' | 'dark'
type ProTheme = 'purple' | 'ocean' | 'forest' | 'midnight'

interface ThemeContextValue {
  mode: ColorMode
  proTheme: ProTheme
  toggleMode: () => void
  setProTheme: (theme: ProTheme) => void
}

const ThemeContext = createContext<ThemeContextValue | null>(null)

export function ThemeProvider({ children }: { children: ReactNode }) {
  const [mode, setMode] = useState<ColorMode>(() => {
    return (localStorage.getItem('theme-mode') as ColorMode) ?? 'light'
  })
  const [proTheme, setProThemeState] = useState<ProTheme>(() => {
    return (localStorage.getItem('theme-pro') as ProTheme) ?? 'purple'
  })

  useEffect(() => {
    const root = document.documentElement
    root.classList.toggle('dark', mode === 'dark')
    root.setAttribute('data-theme', proTheme)
    localStorage.setItem('theme-mode', mode)
  }, [mode, proTheme])

  useEffect(() => {
    localStorage.setItem('theme-pro', proTheme)
  }, [proTheme])

  const toggleMode = () => setMode(m => m === 'light' ? 'dark' : 'light')

  const setProTheme = (theme: ProTheme) => {
    setProThemeState(theme)
  }

  return (
    <ThemeContext.Provider value={{ mode, proTheme, toggleMode, setProTheme }}>
      {children}
    </ThemeContext.Provider>
  )
}

export function useTheme() {
  const ctx = useContext(ThemeContext)
  if (!ctx) throw new Error('useTheme must be used within ThemeProvider')
  return ctx
}

export type { ColorMode, ProTheme }
```

```tsx
          {mode === 'dark' ? '☀️ Claro' : '🌙 Oscuro'}
        </button>
      </div>

      {/* Theme picker */}
      <div>
        <p className="text-sm mb-3" style={{ color: 'var(--color-on-surface-muted)' }}>
          Tema de color
          {!isPremium && <span className="ml-2 text-xs px-2 py-0.5 rounded-full bg-[var(--color-tertiary)] text-white">Pro</span>}
        </p>
        <div className="grid grid-cols-2 gap-3">
          {themes.map(theme => (
            <button
              key={theme.id}
              onClick={() => (!theme.isPro || isPremium) && setProTheme(theme.id as ProTheme)}
              className={`card p-3 flex items-center gap-2 transition-all ${
                proTheme === theme.id ? 'ring-2 ring-[var(--color-primary)]' : ''
              } ${theme.isPro && !isPremium ? 'opacity-50 cursor-not-allowed' : ''}`}
            >
              <div className="w-6 h-6 rounded-full flex-shrink-0" style={{ backgroundColor: theme.color }} />
              <span className="text-sm font-medium">{theme.name}</span>
              {theme.isPro && !isPremium && <span className="ml-auto text-xs">🔒</span>}
            </button>
          ))}
        </div>
      </div>
    </div>
  )
}
```
